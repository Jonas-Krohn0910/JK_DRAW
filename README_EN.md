# JK_DRAW
JK_DRAW is a mathematical visualization tool designed for marine engineers, mechanical engineers, and electrical engineers. The program makes it easy to plot vector functions, analyze technical relationships, and perform calculations commonly used in engineering studies and practical work.
README – JK Draw

Welcome to JK Draw!
This program contains three tools:

    Function Fit (regression)

    Function Plotter (plot functions and analyze graphs)

    Vector Lab (work with vectors)

How to start the program:

    Run setup.bat the first time to install the required Python packages

    When the installation is complete, run start.bat to open the program

Folder structure:

    src/ (contains the program files – do not modify these)

    start.bat (used to start the program)

    setup.bat (used for installation)

    README.txt (this file)

If you experience problems:
The program does not start:

    Make sure you have run setup.bat first

    Check that you have not moved or renamed the src folder

    Try restarting your computer and run start.bat again

Python errors in the window:

    This is often caused by missing packages

    Run setup.bat again to reinstall them

The program opens, but graphs do not appear:

    Check that you have not modified anything in the src folder

    Make sure you have not deleted matplotlib or numpy

Function Fit cannot open Excel files:

    Make sure the file is .xlsx or .xls

    Check that the file is not open in Excel at the same time

Function Plotter shows strange graphs:

    Check that you entered the coefficients correctly in the pop-up window

    Try adjusting the plot interval

If nothing works:

    Delete the entire folder and extract the ZIP file again

    Run setup.bat, then start.bat
