FUNKTIONSTEGNER
--------------------------------------

Generelt:
- Tilføj funktioner via pop-up vinduer (lineær, polynomiel, trigonometrisk, eksponentiel m.fl.)
- Alle funktioner navngives automatisk: f(x), g(x), h(x) …
- Grafer vises i et koordinatsystem med tydelige akser
- Zoom, pan og reset findes i værktøjslinjen over grafen
- UI-størrelse kan justeres (80–180%) for forskellige skærme
- Maksimalt antal funktioner: z(x) er den sidste

Lineær funktion:
- Formel: f(x) = a·x + b
- Bruges til rette linjer og simple trends
- Pop-up kræver: a og b

2. grads funktion:
- Formel: f(x) = a·x² + b·x + c
- Har et toppunkt (maksimum eller minimum)
- Bruges til buede forløb
- Pop-up kræver: a, b og c

3. grads funktion:
- Formel: f(x) = a·x³ + b·x² + c·x + d
- Bruges til S-formede kurver
- Pop-up kræver: a, b, c og d

4. grads funktion:
- Formel: f(x) = a·x⁴ + b·x³ + c·x² + d·x + e
- Bruges til komplekse kurver med flere knæk
- Pop-up kræver: a, b, c, d og e

Eksponentiel funktion:
- Formel: f(x) = a·e^(b·x)
- Bruges når noget vokser eller aftager hurtigt
- Pop-up kræver: a og b

Logaritmisk funktion:
- Formel: f(x) = a·ln(x) + b
- Kræver at x > 0
- Bruges når noget stiger hurtigt og flader ud
- Pop-up kræver: a og b

Potensfunktion:
- Formel: f(x) = a·x^b
- Kræver x > 0 hvis b ikke er et helt tal
- Bruges til vækst der følger en potens
- Pop-up kræver: a og b

Trigonometriske funktioner:
- sin: f(x) = a·sin(b·x)
- cos: f(x) = a·cos(b·x)
- Bruges til bølger og periodiske mønstre
- Pop-up kræver: a og b

Skæringspunkter:
- Finder alle punkter hvor to funktioner krydser hinanden
- Punkterne vises på grafen med koordinater

Akse-skæringer:
- X-akse skæring: f(x) = 0
- Y-akse skæring: f(0)
- Punkterne markeres på grafen

Toppunkt:
- Gælder kun 2. grads funktioner
- Finder både maksimum og minimum i det valgte interval
- Punkterne markeres på grafen

Plot-interval:
- Justér x- og y-intervallet
- Tryk “Opdater akser” for at opdatere visningen
